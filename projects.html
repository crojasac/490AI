Project 1: Word Cloud
We used students' "dream objectives about the class" from an AI course to generate a word cloud. This visualization highlights the most common aspirations and interests among the students.
Python Code Generated Word Cloud (PNG)

Python Code Snippet

# Example: wordcloud.py
  import matplotlib.pyplot as plt
  from wordcloud import WordCloud

  text = "AI class dream objectives..."
  wordcloud = WordCloud().generate(text)
  plt.imshow(wordcloud, interpolation='bilinear')
  plt.axis("off")
  plt.savefig("wordcloud.png")
